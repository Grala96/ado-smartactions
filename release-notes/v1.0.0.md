# v1.0.0 – Initial Release

---

## 🎉 First public release of ADO SmartActions

### 🔀 Branch Name Generator
- Automatically generates a properly formatted Git branch name directly from an open Azure DevOps work item with a single button click.
- The `🔀 Generate branch name` button is injected into the *Create a branch* modal in Azure DevOps.
- Works both on direct work item pages (`/workitems/edit/`) and when the modal is opened from the Sprints board or other views.
- Branch name is built from configurable parts: **prefix** (e.g. `feature`, `bugfix`), **work item number**, and a **slugified title**.
- Automatically fills the branch name input field inside the ADO dialog; falls back to clipboard copy if the field is not available.
- Full support for **Chrome** (Manifest V3) and **Firefox** (Manifest V2).

### ⚙️ Options Page
- Extension settings page allows customization of the branch name format:
  - Type-to-prefix mapping (e.g. *User Story* → `feature`, *Bug* → `bugfix`, *Task* → `task`).
  - Separator character between branch name parts.
  - Toggle to include/exclude each part (`Type`, `Number`, `Title`) and reorder them.
  - **Custom fallback prefix** for work item types not on the whitelist.
  - **Live preview** of the generated branch name — updates instantly while configuring part order and separators.

### 🛠️ Build System
- Single shared source (`src/`) compiled to browser-specific directories (`chrome/`, `firefox/`) via `build.sh`.
