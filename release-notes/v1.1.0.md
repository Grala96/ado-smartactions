# v1.1.0 – Support for `visualstudio.com` domain

---

## What's new

### 🌐 Support for `*.visualstudio.com` domain
- The extension now works with both Azure DevOps hosting variants:
  - `https://dev.azure.com/YOUR-ORG/...`
  - `https://YOUR-ORG.visualstudio.com/...`
- The URL pattern matching in the content script has been updated to recognize both domains.
- `manifest.json` for Chrome and Firefox extended with `https://*.visualstudio.com/*` to allow the content script to run on the legacy domain.
